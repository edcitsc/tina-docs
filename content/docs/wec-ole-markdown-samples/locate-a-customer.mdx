# Locate a Customer

## Applies To

- Utilities: MERC, MGU, NSG, PGL, WPS, WE
- Audience: Account Management, Business Solutions Center, Complex - Residential Billing, Contact Center, Credit - Collections, Field Ops, Payment Processing

## Description

A customer is located in Open-cIS by entering search criteria on the Customer Search window. A customer can be located using various search criteria and is performed as the first step in viewing, modifying, or adding a customer.

## Process

### CIS System Steps - Only

1. On the Home tab, select Customer.
   The Customer Search window displays.
2. Select the appropriate Search Criteria method.

    > Note: Search by using one or more of steps 3 to 6.

3. Do the following to locate a customer by customer identification number (ID), if known:

    - Enter customer identification number in the Customer Id field.
    - Select Search.

    The Customer Details window displays.

    > Note: No other information is necessary if the correct customer ID number is entered. Exit procedure.

4. Do one or more of the following to locate a customer by customer information:

    > Note: If the complete customer name is not known, use the wildcard (*).

    - Select customer type from the Customer Type drop-down list.
    - Enter customer social security number in the Social Security # field.
    - Enter customer phone number in the Phone # field.
    - Enter customer last name in the Last Name field.
    - Enter customer first name in the First Name field.
    - Enter customer middle name in the Middle Name field.

5. Do one or more of the following to locate a customer by business information:

    - Enter full business name in the Customer Full Name field.
    - Select the appropriate business type from the Customer Type drop-down list.
    - Enter business tax identification number in the Tax ID field.
    - Enter business phone number in the Phone # field.
    - Enter doing business as name in the Doing Business As field.

6. Do the following to locate a customer by customer attributes:

    - Select Attributes Expand/Collapse arrow.
    - Select the Add + icon in the Attribute group box.
    - The Attribute Search Criteria window displays.
    - Select the appropriate entity from the Entity Type drop-down list.
    - Select the appropriate subtype from the Sub Type drop-down list.
    - Select the appropriate company from the Company drop-down list.
    - Select the appropriate value from the Attribute Value field.
    - Repeat steps for each applicable attribute.

7. Select Search.
   A list of customers matching the search criteria displays in the Results group box.
